# xmemcached-spring-boot-starter
Spring Boot Starter for XMemcached

## 特性
- spring-boot-starter
- 全局默认过期时间配置
- 自定重连
- 自定义过期时间注解 `Expired` 支持 SpEl 表达式动态配置时间


## Getting Started
```xml
<dependency>
    <groupId>cn.tisson</groupId>
    <artifactId>xmemcached-spring-boot-starter</artifactId>
    <version>${version}</version>
</dependency>
```
